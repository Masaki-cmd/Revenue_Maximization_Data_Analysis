# Revenue_Maximization_Data_Analysis

## プロジェクト概要

本プロジェクトは、提供された顧客、注文、月次収益、および製品サマリーデータを用いて、企業の収益最大化とコスト最適化のためのデータ分析を実施します。データクレンジングから始まり、探索的データ分析（EDA）、課題定義、仮説設定、RFM分析、そして具体的な提案に至るまでのプロセスを詳細に記録しています。

## 分析の目的

*   企業の収益構造を理解し、成長機会を特定する。
*   製品ポートフォリオのパフォーマンスを評価し、改善点を見つける。
*   顧客行動のパターンを分析し、ターゲットマーケティング戦略を策定する。
*   データに基づいた具体的なビジネス提案を行い、収益の最大化およびコストの最適化を支援する。

## データセット

以下の4つのCSVファイルを使用しています。

1.  `customers.csv`: 顧客に関する情報（顧客ID、国、年齢、性別、会員ティア、登録日、総注文数、総支出額、平均注文額、最終購入からの日数、好みカテゴリ、好みデバイス、支払い方法、獲得チャネル、レビュー数、平均レビュー評価、返品数、ウィッシュリストアイテム数、ニュースレター購読状況、チャーン状況）。
2.  `orders.csv`: 注文に関する情報（注文ID、顧客ID、注文日、製品名、カテゴリ、単価、数量、小計、割引率、割引額、送料、税率、税額、合計金額、支払い方法、使用デバイス、配送日数、配送日、注文ステータス、返品状況、顧客評価、セッション時間、購入前閲覧ページ数、リピーターかどうか）。
3.  `monthly_revenue.csv`: 月次の収益サマリー（年、月、四半期、注文数、収益、平均注文額、平均割引率、返品率、ユニーク顧客数、新規顧客数）。
4.  `product_summary.csv`: 製品ごとのサマリー情報（カテゴリ、製品名、総注文数、総収益、平均価格、平均評価、返品率、平均割引率、平均配送日数）。

## 分析プロセス

### 1. データ読み込みと初期確認

各CSVファイルをPandas DataFrameとして読み込み、基本的な情報（データ型、欠損値、先頭行など）を確認しました。

### 2. データクレンジングと前処理

データの正確な分析のため、以下のクレンジングと前処理を実施しました。

*   **`customers_df`**: `registration_date`を`datetime`型に変換。
*   **`orders_df`**: `order_date`と`delivery_date`を`datetime`型に変換。`customer_rating`の欠損値（15749件）は、評価がないことを示すためNaNとして保持し、必要に応じて対応を検討する方針としました。
*   **データの統合**: 顧客ごとの購買行動と属性を合わせて分析するため、`orders_df`と`customers_df`を`customer_id`をキーとして結合し、`merged_df`を作成しました。
*   **日本語表示対応**: グラフ描画における日本語表示の問題を解決するため、`japanize_matplotlib`を導入しました。

### 3. 探索的データ分析 (EDA) と仮説設定

#### 3.1. 月次収益トレンド分析

*   `monthly_revenue_df`を用いて時系列での収益推移を可視化しました。収益は比較的安定しているものの、大幅な成長トレンドや明確な季節性は見られませんでした。
*   **仮説**: 安定した収益基盤を維持しつつ、新たな成長ドライバーを見つける必要がある。

#### 3.2. 製品パフォーマンス分析

*   `product_summary_df`を用いてカテゴリごとの収益貢献度、注文数、返品率などを分析しました。
*   `Electronics`が圧倒的な収益を上げている主要カテゴリであることが判明しました。
*   一方で、`Travel & Luggage`や`Pet Supplies`など一部カテゴリでは返品率が高い傾向が見られました。
*   **仮説1 (収益最大化)**: 高収益カテゴリの製品ラインナップ強化やバンドル販売で、さらなる収益増加が見込める。
*   **仮説2 (コスト最適化)**: 返品率が高いカテゴリの製品について、品質見直し、商品説明改善、顧客サポート強化で返品コストを削減できる。

#### 3.3. 顧客行動分析

統合された`merged_df`を用いて、以下の分析を行いました。

*   **年齢層別分析**: 20-39歳が高支出傾向にあり、60代以上の支出は控えめであることが確認されました。
*   **会員ティア別分析**: PlatinumやGold会員は平均支出額および平均注文額がFree会員と比較して大幅に高く、非常に価値の高い顧客セグメントであることが確認されました。
*   **獲得チャネル別分析**: Paid Ad、Direct、Referralが平均総支出額が高い顧客を獲得しており、効率的なチャネルである可能性が高いことが示唆されました。Social Mediaからの顧客は平均支出額が低い傾向が見られました。
*   **仮説1 (収益最大化)**: 高支出の年齢層向けプロモーション、Gold/Platinum会員へのアップセル・インセンティブ強化で収益増加が見込める。支出が少ない年齢層向けにニーズに合った製品展開も有効。
*   **仮説2 (コスト最適化・収益最大化)**: 高効率チャネルへのマーケティング投資を増やし、低効率チャネルからの顧客には獲得後の育成プログラムを強化することでLTVを向上できる。

#### 3.4. RFM分析による顧客セグメンテーション

Recency（最終購入からの日数）、Frequency（購入頻度）、Monetary（購入金額）を算出し、各指標を5段階でスコアリングすることで、顧客を以下のセグメントに分類しました。

*   **Champions (優良顧客)**
*   **Loyal Customers (優良リピーター)**
*   **Potential Loyalists (優良予備軍)**
*   **New Customers (新規顧客)**
*   **At Risk (離反リスク)**
*   **Lost (離反顧客)**

各セグメントの分布と平均RFM指標を可視化・分析し、顧客行動に基づいて明確に分類することで、高価値顧客の維持、休眠顧客の再活性化、新規顧客の育成といった、よりパーソナライズされた顧客戦略を効果的に実施するための基礎を築きました。

### 4. 総合的な洞察と提案

これまでの分析結果を統合し、収益の最大化およびコストの最適化に向けた以下の具体的な提案を行いました。

#### 4.1. 収益最大化のための提案

1.  **高価値顧客層 (Platinum/Gold会員、20-39歳) へのエンゲージメント強化**
    *   特別プロモーション、アップセル・クロスセル、ターゲット広告、ウィッシュリスト活用。
2.  **主力製品カテゴリ (`Electronics`など) の戦略的強化**
    *   新製品開発・拡充、バンドル販売、在庫最適化。
3.  **効果的な獲得チャネルへの投資集中と最適化**
    *   広告予算の再配分、Referralプログラムの強化。
4.  **顧客エンゲージメントの向上**
    *   パーソナライズされたレコメンデーション。

#### 4.2. コスト最適化のための提案

1.  **高返品率カテゴリ (`Travel & Luggage`, `Pet Supplies`) の改善**
    *   製品品質の向上、商品説明の明確化、レビューの活用。
2.  **割引戦略の最適化**
    *   ターゲット割引、割引率の継続的モニタリング。
3.  **低効率獲得チャネル (`Social Media`) の見直し**
    *   獲得後のLTV向上施策、チャネル戦略の再評価。

#### 4.3. RFMセグメント別収益最大化のための具体的な施策（詳細版）

各RFMセグメント（Champions, Loyal Customers, Potential Loyalists, New Customers, At Risk / Lost）に対して、それぞれに最適化された詳細なマーケティング戦略と具体的な施策を策定しました。

*   **Champions**: 排他的な体験、パーソナライズされたギフト、フィードバックの優先、紹介プログラム強化。
*   **Loyal Customers**: 継続購入インセンティブ、カテゴリ横断レコメンデーション、ロイヤリティプログラム強化、コミュニティ形成。
*   **Potential Loyalists**: 初回リピート購入促進、商品知識の深化、パーソナルショッパーサービス、限定セールへの早期アクセス。
*   **New Customers**: 充実したオンボーディング、ブランドの価値訴求、限定的な割引と特典、パーソナライズされたフォローアップ。
*   **At Risk / Lost**: 再エンゲージメントキャンペーン、フィードバック収集、新しい価値の提供、顧客属性に応じたメッセージング。
